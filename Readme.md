# Student Records Search Application

## Project Overview

This project is a Student Records Search Application built using HTML, CSS, and JavaScript.

The application displays student information in card format and allows users to search for students by name. The search results update dynamically as the user types in the search box.

## Features

* Display student records using JavaScript arrays and objects.
* Render student information dynamically in card format.
* Search students by name.
* Real-time filtering using the search input field.
* Responsive card layout using Flexbox.
* Fixed navigation bar and search section.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Student Object Structure

Each student record contains:

* Student Name
* Marks
* Class
* Address

Example:

{
Student_Name: "sri",
Marks: "98%",
Class: "12th",
Address: "123 ABC colony"
}

## Functionality

### displayStudents()

* Dynamically renders student cards on the webpage.
* Uses the JavaScript map() method to create HTML content.

### searchStudent()

* Retrieves text entered in the search box.
* Filters student records using the filter() method.
* Displays matching student records in real time.
